     Реализовать функции:

     (@FIRSTN n list) - возвращает в качестве результата список
        (n > 0)        из n первых элементов списка list.


     Пример:
            (@FIRSTN  0 '(A B))    => NIL
            (@FIRSTN  2 '(A B C))  => (A B)
            (@FIRSTN  3 '(A B))    => (A B)

     (@BUTLAST list n) - возвращает в качестве результата список
                        без последних n элементов list. Если n
                        опущено либо < 0, то возвращает список
                        без последнего элемента.

     Примеры:
             (@BUTLAST '(A B C))     => (A B)
             (@BUTLAST '(A B C D) 2) => (A B)


     (@SUBLIST list n m) - выделяет из списка list элементы с n
        n,m >= 0          по m и возвращает их в виде резуль-
          n >= m          тирующего списка.

     Пример:
            (@SUBLIST '(A B C D E F) 2 4) => (C D E)
            (@SUBLIST '(A B C D) 2 2)     => (C)
            (@SUBLIST '(A B C D E F) 2)   => (C D E F)
            (@SUBLIST  (A B C D E F) 03)  => (A B C D)
